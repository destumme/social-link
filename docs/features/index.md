## Agent Instructions: Creating a New Feature

Prompt the user for this information if needed.
Do not design the entire feature, gather information as needed to fill out the template.

1. Create a new file in this directory named `FEAT-NNN-short-title.md` (use the next sequential number).
2. Copy the contents of `TEMPLATE.md` into the new file.
3. Fill in the YAML frontmatter (`page`, `area`, `priority`, `status`, `created`).
4. Write the summary, motivation, scope, and design sections.
5. Add a link to the new file in the **Proposed** table above.
